# Vetx app with kotlin

Simple example of [Vert.x](https://vertx.io/) with [Kotlin](https://kotlinlang.org/).

## Run

1. ``gradle build``
2. ``gradle run``
3. Open [localhost:9090](http://localhost:9090/)
