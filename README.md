# Vetx app with kotlin

Simple example of [Vert.x](https://vertx.io/) with [Kotlin](https://kotlinlang.org/).

## Run

### Backend
- ``gradle build``
- ``gradle run``

### Frontend

- ``npm install`` ``npm run watch``

### Go

1. Open [localhost:9090/home](http://localhost:9090/home)
